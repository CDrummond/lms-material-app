<p>Cette application fournit une interface affichant du contenu web pour piloter une instance de <a href='https://lyrion.org/'>Lyrion Music Server (LMS)</a>.</p><p>Elle nécessite qu'un serveur <a href='https://lyrion.org/'>Lyrion Music Server (LMS)</a> soit présent sur votre réseau et que le plugin <a href='https://github.com/CDrummond/lms-material'>MaterialSkin</a> soit installé sur ce serveur.</p>
